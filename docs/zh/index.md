# 系统安全研究之旅

欢迎来到《系统安全研究之旅》——一本面向系统安全研究方法与实践的综合指南。

## 关于本书

系统安全关注攻击者、漏洞与资产三者之间的关系，通过识别与修补漏洞、构建防护机制来保护关键资产免受攻击。其特点可概括为：基础性、系统性、跨层性。本书聚焦推动系统安全研究前进的方法与实践。

## 你将学到什么

本书分为十个部分，每个部分都聚焦系统安全研究的一个关键方面：

1. **[系统安全研究生存指南](p1-why-system-security.md)** — 理解该领域的动机与挑战
2. **[系统安全研究方法论](p2-Research-Philosophy.md)** — 如何开展系统安全研究
3. **[如何查找相关工作](p3-literature-review.md)** — 导航研究领域
4. **[构建研究知识体系](p4-knowledge-for-research.md)** — 必备知识基础
5. **[系统安全研究的工程实现](p5-Summary-Research-SE.md)** — 连接理论与实现
6. **[研究展示与演讲](p6-research-presentation.md)** — 有效呈现与沟通你的工作
7. **[技术转化与落地实践](p7-technology-transfer.md)** — 从研究到实际影响
8. **[研究与开发中的陷阱、调整与技巧](p8-dont-waste-time.md)** — 生产力和时间管理
9. **[研究团队与实验室管理](p9-research-groups.md)** — 协作研究动态
10. **[繁荣中的危机：内部捷径与外部挫折](p10-properity-of-crisis.md)** — 在挑战中寻找机会

## 本书适合谁

本书适合以下读者：

- **研究生** 准备进入系统安全研究领域，但缺少系统性的指导
- **年轻教师** 想带学生做研究，却不确定如何起步与推进
- **安全从业者** 希望理解研究视角与方法
- **一般读者** 对系统安全的挑战与机遇感到好奇

## 开始阅读

建议从[第一部分：系统安全研究生存指南](p1-why-system-security.md)开始，理解这个领域为何艰难、却又值得投入。
